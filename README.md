### 📝 Tentang Game
**Tic Tac Tobat** adalah koleksi modifikasi game *Tic-Tac-Toe* yang mengintegrasikan algoritma **Alpha-Beta Pruning**. Teknologi ini dirancang untuk menciptakan kecerdasan buatan (AI) yang kompetitif, memberikan pengalaman bermain yang menantang, inovatif, dan menuntut strategi tinggi.

### 🚀 Cara Menjalankan
1.  Buka direktori game dan jalankan file `TicTacTobat.exe`.
2.  Pilih salah satu dari **4 mode permainan** yang tersedia pada menu utama.
3.  Gunakan **mouse** untuk memilih kotak yang ingin diisi.

### 🕹️ Fitur Mode Permainan
| Mode | Deskripsi |
| :--- | :--- |
| **Classic** | Permainan standar melawan AI dengan tingkat kecerdasan optimal. |
| **Infinite** | Strategi dinamis di mana simbol lama akan menghilang setelah jumlah langkah tertentu. |
| **Hidden Trap** | Mode penuh risiko di mana terdapat jebakan tersembunyi di dalam papan. |
| **Clash Mode** | Adu insting menebak langkah lawan; jika terjadi bentrokan pada kotak yang sama, pemenang ditentukan melalui mekanisme suit. |

### ⚙️ Catatan Teknis
* **Pengaturan Aturan:** File `settings.json` akan otomatis dibuat jika Anda mencentang opsi *"Jangan tampilkan aturan lagi"*.
* **Reset Preferensi:** Jika ingin menampilkan kembali panduan aturan di awal permainan, silakan hapus file `settings.json` di folder utama.
