TENTANG GAME:
Tic Tac Tobat adalah koleksi game Tic-Tac-Toe yang dimodifikasi dengan 
algoritma Alpha-Beta Pruning untuk memberikan pengalaman bermain yang 
menantang dan inovatif.

CARA MENJALANKAN:
1. Jalankan file "TicTacTobat.exe".
2. Pilih salah satu dari 4 mode permainan yang tersedia.
3. Gunakan mouse untuk mengklik kotak yang ingin diisi.

FITUR MODE:
- Classic: Game standar melawan AI yang sangat cerdas.
- Infinite: Strategi dinamis di mana simbol lama akan menghilang.
- Hidden Trap: Hati-hati menaruh bidak, salah langkah bisa kena jebakan!
- Clash Mode: Uji insting menebak langkah lawan dan adu suit jika terjadi bentrokan.

CATATAN:
- File 'settings.json' akan otomatis dibuat saat Anda mencentang "Jangan tampilkan aturan lagi".
- Jika ingin memunculkan kembali aturan permainan, hapus file 'settings.json' tersebut.
